
# Mini E-Commerce React App
##Live Demo:
https://92sn3q-3000.csb.app/

## Description
A mini e-commerce application built using React that fetches product data
from a public API and allows users to browse products and manage a cart.

## Features
- Product listing using external API (FakeStore API)
- Search products by name
- Filter products by category
- Sort products by price (Low to High / High to Low)
- Add to cart, update quantity, and remove items
- Cart total price calculation

## Tech Stack
- React (Functional Components)
- JavaScript
- React Hooks
- Context API
- CSS
